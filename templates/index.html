How to Export Your Chat Files

Follow these simple steps to get your chat files ready for analysis. It's easier than you think!

W

WhatsApp Export

1

Open WhatsApp

Go to the specific chat you want to analyze

2

Access Chat Menu

Tap the three dots (⋮) in the top right corner → More → Export Chat

3

Choose Export Option

Select "Without Media" (this keeps the file small and manageable)

4

Save the File

Share the .txt file with yourself via email or save it to your device

You'll get a .txt file ready for upload!

IG

Instagram Export

1

Open Instagram Settings

Go to your profile → Menu → Settings

2

Privacy & Security

Tap Privacy & Security → Download Your Information

3

Request Data

Enter your email address → Select JSON format → Request download

4

Extract & Upload

Instagram will email you a .zip file → Extract it → Find messages.json files

Note: Instagram data requests can take up to 48 hours
